# 学生管理系统 (Student Management System)

## 项目简介
基于Spring Boot开发的学生信息管理系统，支持学生课表查询、个人信息管理、考试成绩查询等核心功能。

## 技术栈
- 后端：Spring Boot、MyBatis、MySQL
- 前端：HTML/CSS/JavaScript
- 构建工具：Maven

## 功能模块
- 学生信息管理
- 课表查询
- 考试成绩管理

## 运行说明
1.  克隆项目：`git clone https://github.com/anlanlp/student_system.git`
2.  配置MySQL数据库连接
3.  通过Maven编译并启动Spring Boot项目
4.  访问前端页面使用系统
